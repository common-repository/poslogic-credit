The Poslogic Credit Gateway plugin adds a payment gateway for buying goods on credit in your online shop

== Description ==

The Poslogic Credit Gateway plugin enables you to sell goods on credit or by installments through the broker Finservice.
After plugin installation and setup your clients will be able to pay for goods in credit.

== Setup ==

For correct operation, you will need a unique identifier, which you can get by contacting us using the contacts indicated on the site [http://finservice.pro.
](http://finservice.pro).
